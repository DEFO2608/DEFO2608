Etiquetas de HTML

HTML es un lenguaje de programación muy divertido y entretenido, requiero como todo de práctica para adquirir destreza en el manejo de sus etiquetas y elementos.



Dentro de las etiquetas que encontrarás en HTML están las siguientes:

  • P: la cual permite añadir el texto que presentaremos en nuestra página web.
  • H1...H6: Son las etiquetas que permitirán añadir los títulos en la página web. H1 es el título de mayor tamaño y H6 el de menor tamaño.
  • UL y OL: son las etiquetas que permiten agregar una lista a la página web. "UL" permite añadir una lista no numerada. "OL" permite añadir una lista numerada.

Propiedades de CSS

CSS es un lenguaje de programación que te permite dar estilo a tu página, por ejemplo, modificar el tipo de letra, el color de la letra, el color de fondo, modificar lo colores de las fuentes en títulos, y hasta dar un efecto 3D a la web.

Dentro de las propiedades de CSS están:

  • Color: permite dar diseño a la fuente, fondo de la página, entre otros.
  • Background-color: permite definir un color de fondo en la página.
  • Font-size: permite modificar el tamaño de la fuente seleccionada dentro de la página web.
